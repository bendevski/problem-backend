Project for mini problem posting system, does not use authentication since it was unneeded for what I wanted.

##Deployment
Make sure to have composer installed.
Navigate to directory
### 'composer install'
### 'touch database/database.sqlite'
### 'php artisan migrate'
### 'php artisan serve'
